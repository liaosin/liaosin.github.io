基于hexo+github pages的个人博客开发笔记(2/2)

Veröffentlicht am 2017-04-16 |

Hexo theme 更换（yilia）

  1. 打开终端，切换路径址至blog中；
  2. 安装主题：$ git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia
  3. 在blog根目录下，打开_config.yml，更改主题：theme: yilia

Hexo 数学公式渲染

  1. 安装MathJax

    1. 打开终端，切换路径至blog中；

    2. 执行以下代码：

      $ npm install hexo-math —save

      $ hexo math install

    3. 偶尔会出现渲染bug，进行以下更改：

      1. 打开marked.js（在./node_modules/marked/lib/中）

      2. 进行两个替换

        escape: /^\([\`*{}[]()# +-.!_>])/,

        escape: /^\([`*[]()# +-.!_>])/,

        em: /^\b_((?:[^_]|__)+?)_\b|^*((?:**|[\s\S])+?)*(?!*)/,
        em:/^*((?:**|[\s\S])+?)*(?!*)/,

Hexo 图片存储

  1. 确认 _config.yml 中有 post_asset_folder:true ；
  2. 打开终端，切换路径至blog中；
  3. 执行：npm install https://github.com/CodeFalling/hexo-asset-image —save
  4. 这样一来，新建一个博客时，hexo将自动新建一个与之同名的文件夹，在图片插入时直接使用![filename](floder/pic.jpg)即可。
